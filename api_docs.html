飞书表格读写服务器

API接口文档 - 点击按钮直接调用

📖

飞书表格读取

GET 读取表格数据 需传参
读取飞书表格指定范围的数据
参数： sheetid, spreadsheettoken, range(可选，默认A:J)
✏️

飞书表格写入

GET 更新指定单元格 需传参
更新飞书表格任意单元格内容
参数： sheetid, spreadsheettoken, cell_range, content
GET 更新流程追踪J列 需传参
更新表格J列（流程追踪）指定行内容
参数： sheetid, spreadsheettoken, row_num, content
GET 更新订单窗口号N列 需传参
更新表格N列（订单窗口号）指定行内容
参数： sheetid, spreadsheettoken, row_num, window_num
📱

手机号队列

GET 加载手机号队列 可点击
从表格253571读取数据到队列
点击调用
GET 取出一条数据 可点击
从队列取出一条数据（FIFO），取出后移除
点击调用
GET 放回数据到队列 需传参
把未消费成功的数据放回队列
参数： row_num, sheetid, spreadsheettoken, phone_number, sms_url
GET 写入窗口名称到F列 需传参
在指定表格F列对应行写入窗口名称
参数： row_num, sheetid, spreadsheettoken, window_name
GET 查看队列状态 可点击
查看手机号队列剩余数量
点击调用
